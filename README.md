## 分布式微服务电商系统
使用Spring Boot和Spring Cloud
## 
+ openfeign: 远程服务调用
+ nacos: 服务注册和配置中心
+ mysql: 数据库
+ Redisson: 分布式锁
+ SpringCache+Redis: 分类和商品数据缓存 
+ SpringSession: 分布式session
+ RabbitMQ: 死信队列定时关闭订单、解锁库存
+ ElasticSearch: 商品搜索
+ Sentinel: 流量控制
+ Sleuth+Zipkin: 链路追踪
+ Alipay：支付服务
